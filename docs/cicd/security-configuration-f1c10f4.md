# Security Configuration

Configure security features in SAP Continuous Integration and Delivery.

-   [**Manage Allowed Spaces**](manage-allowed-spaces-0181fc5.md)

    Prevent unauthorized users from having API access to SAP Continuous Integration and Delivery by maintaining a list of Cloud Foundry spaces that are allowed to request service instances.

-   [**Configure a Credential List**](configure-a-credential-list-907b44a.md)

    Specify the credentials that are passed to the build in repository-configured SAP Continuous Integration and Delivery jobs.
